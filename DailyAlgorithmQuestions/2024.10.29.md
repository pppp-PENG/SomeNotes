[3211. 生成不含相邻零的二进制字符串](https://leetcode.cn/problems/generate-binary-strings-without-adjacent-zeros/)

给你一个正整数 `n`。

如果一个二进制字符串 `x` 的所有长度为 2 的子字符串中包含 **至少** 一个 `"1"`，则称 `x` 是一个 **有效** 字符串。

返回所有长度为 `n` 的 **有效** 字符串，可以以任意顺序排列。

```
输入：n = 3
输出：["010","011","101","110","111"]

输入：n = 1
输出：["0","1"]
```

**提示：**

- `1 <= n <= 18`



### 题解

题目等价于寻找所有不含连续0的字符串

`n` 小于等于 `18`，可以直接用递归完成

时间复杂度：$O(2^n)$

空间复杂度：$O(2^n)$

```java
class Solution {
    List<String> list = new ArrayList<String>();

    public List<String> validStrings(int n) {
        StringBuilder sb = new StringBuilder();
        x(n, sb, 0);
        return list;
    }

    public void x(int n, StringBuilder sb, int num) {
        if(num == n) {
            list.add(sb.toString());
            return;
        }
        if(num == 0 || sb.charAt(num - 1) != '0') {
            sb.append('1');
            x(n, sb, num + 1);
            sb.deleteCharAt(num);
            sb.append('0');
            x(n, sb, num + 1);
            sb.deleteCharAt(num);
        }
        else {
            sb.append('1');
            x(n, sb, num + 1);
            sb.deleteCharAt(num);
        }
        return;
    }
}
```
