[685. 冗余连接 II](https://leetcode.cn/problems/redundant-connection-ii/)

在本问题中，有根树指满足以下条件的 **有向** 图。该树只有一个根节点，所有其他节点都是该根节点的后继。该树除了根节点之外的每一个节点都有且只有一个父节点，而根节点没有父节点。

输入一个有向图，该图由一个有着 `n` 个节点（节点值不重复，从 `1` 到 `n`）的树及一条附加的有向边构成。附加的边包含在 `1` 到 `n` 中的两个不同顶点间，这条附加的边不属于树中已存在的边。

结果图是一个以边组成的二维数组 `edges` 。 每个元素是一对 `[ui, vi]`，用以表示 **有向** 图中连接顶点 `ui` 和顶点 `vi` 的边，其中 `ui` 是 `vi` 的一个父节点。

返回一条能删除的边，使得剩下的图是有 `n` 个节点的有根树。若有多个答案，返回最后出现在给定二维数组的答案。

```
输入：edges = [[1,2],[1,3],[2,3]]
输出：[2,3]

输入：edges = [[1,2],[2,3],[3,4],[4,1],[1,5]]
输出：[4,1]
```

**提示：**

- `n == edges.length`
- `3 <= n <= 1000`
- `edges[i].length == 2`
- `1 <= ui, vi <= n`



### 题解

分类讨论

1. 根节点有了父节点，那么每个节点都有父节点，必然会形成环

2. 根节点唯一，但出现了双父节点的节点（记双父节点对应的边分别为a边、b边）

    2.1 图中出现有向环，那么必须输出环内的边（此时a、b边有且只有一条在环内）

    2.2 图中无有向环，那么输出最后出现的边

解题过程：

遍历——随时判断是否成环、是否有节点有两个父节点，若出现则记住该边、节点

判断——若遍历阶段无双父节点存在，则为第1种情况；若出现双父节点，则为第2种情况

时间复杂度：$O(n)$

空间复杂度：$O(n)$

```java
class Solution {
	/*
		存储好父节点，画好有向图，遍历过程中随时判断是否出现环，若出现环则存储该边
		1. 若不存在双父节点的节点，则输出一开始存储的边
		2. 存在有双父节点的节点，则输出这两条边构成环的那一边
	*/
	public int[] findRedundantDirectedConnection(int[][] edges) {
        int n = edges.length;
        int[] parent = new int[n + 1];
        int parentTmp = -1, son = 0;
        int[] circleEdge = new int[2];
        for(int[] edge : edges) {
            if(parent[edge[1]] != 0) {
                parentTmp = parent[edge[1]];
                son = edge[1];
            }
            parent[edge[1]] = edge[0];
            if(isCircle(parent, edge[1], edge[1], new HashSet()))
                circleEdge = edge;
        }
        if(parentTmp == -1) return circleEdge;
        if(isCircle(parent, son, son, new HashSet())) return new int[] {parent[son], son};
        int p = parent[son];
        parent[son] = parentTmp;
        parentTmp = p;
        if(isCircle(parent, son, son, new HashSet())) return new int[] {parent[son], son};
        return new int[] {parentTmp, son};
    }

	public boolean isCircle(int[] parent, int idx, int tmp, Set set) {
		if(parent[idx] == 0) return false;
        if(parent[idx] == tmp) return true;
        set.add(idx);
        if(set.contains(parent[idx])) return false;
        return isCircle(parent, parent[idx], tmp, set);
    }
}
```
