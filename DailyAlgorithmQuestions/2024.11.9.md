[3242. 设计相邻元素求和服务](https://leetcode.cn/problems/design-neighbor-sum-service/)

给你一个 `n x n` 的二维数组 `grid`，它包含范围 $[0, n^2 - 1]$ 内的**不重复**元素。

实现 `neighborSum` 类：

- `neighborSum(int [][]grid)` 初始化对象。
- `int adjacentSum(int value)` 返回在 `grid` 中与 `value` 相邻的元素之**和**，相邻指的是与 `value` 在上、左、右或下的元素。
- `int diagonalSum(int value)` 返回在 `grid` 中与 `value` 对角线相邻的元素之**和**，对角线相邻指的是与 `value` 在左上、右上、左下或右下的元素。

```
输入：
["neighborSum", "adjacentSum", "adjacentSum", "diagonalSum", "diagonalSum"]
[[[[0, 1, 2], [3, 4, 5], [6, 7, 8]]], [1], [4], [4], [8]]
输出：[null, 6, 16, 16, 4]

输入：
["neighborSum", "adjacentSum", "diagonalSum"]
[[[[1, 2, 0, 3], [4, 7, 15, 6], [8, 9, 10, 11], [12, 13, 14, 5]]], [15], [9]]
输出：[null, 23, 45]
```

**提示：**

- $3 <= n == grid.length == grid[0].length <= 10$
- $0 <= grid[i][j] <= n^2 - 1$
- 所有 $grid[i][j]$ 值均不重复。
- `adjacentSum` 和 `diagonalSum` 中的 `value` 均在范围 $[0, n^2 - 1]$ 内。
- 最多会调用 `adjacentSum` 和 `diagonalSum` 总共 $2 * n^2$ 次。



### 题解

既然需要多次调用 `adjacentSum` 和 `diagonalSum` 方法，那可以提前（即在构造函数时）将这些方法需要的返回值存好

具体来说，可以定义 `aSum`  和 `dSum` 两个二维数组，分别存对应位置元素调用 `adjacentSum` 和 `diagonalSum` 方法的返回值

即：若 $grid[i][j]=value$，

​	则 $adjacentSum(value)->aSum[i][j]$，$diagonalSum(value)->dSum[i][j]$

对于 `aSum`  和 `dSum` 的构造，只需在构造函数时遍历一次 `grid`

- `grid` 属性：存储原二维数组
- `aSum` 属性：存储`grid` 数组每个元素对应的相邻的元素之和
- `dSum` 属性：存储`grid` 数组每个元素对应的对角线相邻的元素之和
- `NeighborSum` 构造方法：创建该二维数组，并调用 `add` 方法
- `adjacentSum` 方法：返回对应的 `aSum` 值
- `diagonalSum` 方法：返回对应的 `dSum` 值
- `add` 方法：用该元素更新 `aSum`  和 `dSum` 属性

时间复杂度：$O(n^2)$

空间复杂度：$O(n^2)$

```java
class NeighborSum {
    int[][] grid;
    int[][] aSum;
    int[][] dSum;
    int n;

    public NeighborSum(int[][] grid) {
        this.grid = grid;
        this.n = grid.length;
        aSum = new int[this.n][this.n];
        dSum = new int[this.n][this.n];
        for(int i = 0; i < n; i ++)
            for(int j = 0; j < n; j ++)
                add(i, j, grid[i][j]);
    }
    
    public int adjacentSum(int value) {
        for(int i = 0; i < n; i ++)
            for(int j = 0; j < n; j ++)
                if(grid[i][j] == value)
                    return this.aSum[i][j];
        return -1;
    }
    
    public int diagonalSum(int value) {
        for(int i = 0; i < n; i ++)
            for(int j = 0; j < n; j ++)
                if(grid[i][j] == value)
                    return this.dSum[i][j];
        return -1;
    }

    public void add(int i, int j, int value) {
        for(int x = -1; x < 2; x ++) {
            for(int y = -1; y < 2; y ++) {
                if(x == 0 && y == 0)
                    continue;
                int a = i + x, b = j + y;
                if(a >= 0 && a < n && b >= 0 && b < n) {
                    if((x + y + 4) % 2 == 0)
                        this.dSum[a][b] += value;
                    else
                        this.aSum[a][b] += value;
                }
            }
        }
    }
}

/**
 * Your NeighborSum object will be instantiated and called as such:
 * NeighborSum obj = new NeighborSum(grid);
 * int param_1 = obj.adjacentSum(value);
 * int param_2 = obj.diagonalSum(value);
 */
```
