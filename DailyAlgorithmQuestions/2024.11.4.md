[633. 平方数之和](https://leetcode.cn/problems/sum-of-square-numbers/)

给定一个非负整数 `c` ，你要判断是否存在两个整数 `a` 和 `b`，使得  $a^2 + b^2 = c$ 。

```
输入：c = 5
输出：true

输入：c = 3
输出：false
```

**提示：**

- $0 <= c <= 2^{31} - 1$



### 题解

由题可知，不妨设 $a \le b \le \sqrt c$，则可以利用双指针暴力枚举，$a$ 从 $0$ 开始，而 $b$ 从 $\sqrt c$ 开始

时间复杂度：$O(\sqrt c)$

空间复杂度：$O(1)$

```java
class Solution {
    public boolean judgeSquareSum(int c) {
        long p = 0, q = (long)Math.sqrt(c);
        long tmp;
        while(p <= q) {
            tmp = p * p + q * q; // 该运算可能会爆int，可提前将p、q设为long类型
            if(tmp == c) return true;
            else if(tmp < c) p ++;
            else q --;
        }
        return false;
    }
}
```
