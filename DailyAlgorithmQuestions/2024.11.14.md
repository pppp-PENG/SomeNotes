[3249. 统计好节点的数目](https://leetcode.cn/problems/count-the-number-of-good-nodes/)

现有一棵 **无向** 树，树中包含 `n` 个节点，按从 `0` 到 `n - 1` 标记。树的根节点是节点 `0` 。给你一个长度为 `n - 1` 的二维整数数组 `edges`，其中 `edges[i] = [ai, bi]` 表示树中节点 `ai` 与节点 `bi` 之间存在一条边。

如果一个节点的所有子节点为根的子树包含的节点数相同，则认为该节点是一个 **好节点**。

返回给定树中 **好节点** 的数量。

**子树** 指的是一个节点以及它所有后代节点构成的一棵树。

```
输入：edges = [[0,1],[0,2],[1,3],[1,4],[2,5],[2,6]]
输出：7

输入：edges = [[0,1],[1,2],[2,3],[3,4],[0,5],[1,6],[2,7],[3,8]]
输出：6

输入：edges = [[0,1],[1,2],[1,3],[1,4],[0,5],[5,6],[6,7],[7,8],[0,9],[9,10],[9,12],[10,11]]
输出：12
```

**提示：**

- $2 <= n <= 10^5$
- `edges.length == n - 1`
- `edges[i].length == 2`
- `0 <= ai, bi < n`
- 输入确保 `edges` 总表示一棵有效的树。



### 题解

害怕递归爆栈（根据官方题解来看好像并不会），所以采用 `HashMap` 存储无向图 + 直接循环的方法

分三步：

1. 用 `HashMap` 存储所有边的信息，`value` 值为与节点编号为 `key` 值的节点相连接的所有节点
2. 从编号为 `0` 的节点开始广度优先搜索该无向图，并用队列存储好搜索顺序（应对输入乱序的情况）
3. 从队列尾向前遍历，依次计算每个节点的重量（所有递归子节点数），并计数好节点个数

时间复杂度：$O(n)$

空间复杂度：$O(n)$

```java
class Solution {
    public int countGoodNodes(int[][] edges) {
        int n = edges.length;
        // first step
        HashMap<Integer, ArrayList<Integer>> map = new HashMap<>();
        for(int[] edge : edges) {
            if(!map.containsKey(edge[0]))
                map.put(edge[0], new ArrayList<Integer>());
            if(!map.containsKey(edge[1]))
                map.put(edge[1], new ArrayList<Integer>());
            ArrayList<Integer> tmp1 = map.get(edge[0]);
            ArrayList<Integer> tmp2 = map.get(edge[1]);
            tmp1.add(edge[1]);
            tmp2.add(edge[0]);
            map.put(edge[0], tmp1);
            map.put(edge[1], tmp2);
        }
        // second step
        int[] nodes = new int[n + 1];
        boolean[] flag = new boolean[n + 1];
        nodes[0] = 0;
        flag[0] = true;
        for(int i = 0, j = 1; i < n + 1; i ++) {
            if(map.get(nodes[i]) == null)
                continue;
            ArrayList<Integer> tmp = new ArrayList<>();;
            for(int v : map.get(nodes[i])) {
                if(flag[v])
                    continue;
                tmp.add(v);
                nodes[j] = v;
                flag[v] = true;
                j ++;
            }
            if(tmp.size() == 0)
                map.put(nodes[i], null);
            else
                map.put(nodes[i], tmp);
        }
        // third step
        int res = 0;
        int[] weight = new int[n + 1];
        for(int i = n; i >= 0; i --) {
            // if leaf
            if(map.get(nodes[i]) == null) {
                weight[nodes[i]] = 1;
                res ++;
            }
            else {
                boolean f = true;
                int sum = 1, tmp = weight[map.get(nodes[i]).get(0)];
                for(int s : map.get(nodes[i])) {
                    if(weight[s] != tmp)
                        f = false;
                    sum += weight[s];
                }
                if(f)
                    res ++;
                weight[nodes[i]] = sum;
            }
        }
        return res;
    }
}
```
