[1338. 数组大小减半](https://leetcode.cn/problems/reduce-array-size-to-the-half/)

给你一个整数数组 `arr`。你可以从中选出一个整数集合，并删除这些整数在数组中的每次出现。

返回 **至少** 能删除数组中的一半整数的整数集合的最小大小。

```
输入：arr = [3,3,3,3,5,5,5,2,2,7]
输出：2

输入：arr = [7,7,7,7,7,7]
输出：1
```

**提示：**

- $1 <= arr.length <= 10^5$
- `arr.length` 为偶数
- $1 <= arr[i] <= 10^5$



### 题解

枚举

先统计数组中每个整数分别出现了几次

再将结果从最高出现次数到最低出现次数进行排序

最后从最高出现次数的整数开始进行删除

删除元素大于等于一半时停止返回删除的整数个数

时间复杂度：$O(n)$

空间复杂度：$O(n)$

```java
class Solution {
    public int minSetSize(int[] arr) {
        int[] cnt = new int[100005];
        for(int e : arr) cnt[e] ++;
        int[] num = new int[100005];
        for(int c : cnt) num[c] ++;
        int n = arr.length;
        int sum = 0, res = 0;
        for(int i = 100000; i > 0; i --) {
            if(num[i] == 0) continue;
            for(int j = 0; j < num[i]; j ++) {
                sum += i;
                res ++;
                if(sum * 2 >= n) return res;
            }
        }
        return res;
    }
}
```
