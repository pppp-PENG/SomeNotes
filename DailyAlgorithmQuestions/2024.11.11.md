[1547. 切棍子的最小成本](https://leetcode.cn/problems/minimum-cost-to-cut-a-stick/)

有一根长度为 `n` 个单位的木棍，棍上从 `0` 到 `n` 标记了若干位置。例如，长度为 **6** 的棍子可以标记如下：

![img](https://assets.leetcode-cn.com/aliyun-lc-upload/uploads/2020/08/09/statement.jpg)

给你一个整数数组 `cuts` ，其中 `cuts[i]` 表示你需要将棍子切开的位置。

你可以按顺序完成切割，也可以根据需要更改切割的顺序。

每次切割的成本都是当前要切割的棍子的长度，切棍子的总成本是历次切割成本的总和。对棍子进行切割将会把一根木棍分成两根较小的木棍（这两根木棍的长度和就是切割前木棍的长度）。请参阅第一个示例以获得更直观的解释。

返回切棍子的 **最小总成本** 。

```
输入：n = 7, cuts = [1,3,4,5]
输出：16

输入：n = 9, cuts = [5,6,1,4,2]
输出：22
```

**提示：**

- $2 <= n <= 10^6$
- `1 <= cuts.length <= min(n - 1, 100)`
- `1 <= cuts[i] <= n - 1`
- `cuts` 数组中的所有整数都 **互不相同**



### 题解

动态规划求解

`dp[i][j]` 表示执行第 `i` 个 `cut` 到第 `j` 个 `cut` 的最小总成本，`cutsN` 表示 `cuts` 数组的长度

初始状态：`dp` 数组初始为全 `0`

状态转移：`dp[i][j] = min{dp[i][k - 1] + dp[k + 1][j]} + (cuts[j + 1] - cuts[i - 1])`，其中 $k\in[i,j]$

结果状态：`dp[1][cutsN]`

时间复杂度：$O((cutsN)^3)$

空间复杂度：$O((cutsN)^2)$

```java
class Solution {
    /**
        dp[i][j] = min{dp[i][k - 1] + dp[k + 1][j]} + (cuts[j + 1] - cuts[i - 1])
     */
    public int minCost(int n, int[] cuts) {
        int cutsN = cuts.length;
        Arrays.sort(cuts);
        int[] cutting = new int[cutsN + 2];
        cutting[0] = 0;
        cutting[cutsN + 1] = n;
        for(int i = 0; i < cutsN; i ++)
            cutting[i + 1] = cuts[i];
        int[][] dp = new int[cutsN + 2][cutsN + 2];
        for(int d = 0; d < cutsN; d ++) {
            for(int i = 1; i <= cutsN - d; i ++) {
                int j = i + d;
                int min = 9999999;
                for(int k = i; k <= j; k ++)
                    min = Math.min(dp[i][k - 1] + dp[k + 1][j], min);
                dp[i][j] = min + (cutting[j + 1] - cutting[i - 1]);
            }
        }
        return dp[1][cutsN];
    }
}
```
