[3258. 统计满足 K 约束的子字符串数量 I](https://leetcode.cn/problems/count-substrings-that-satisfy-k-constraint-i/)

给你一个 **二进制** 字符串 `s` 和一个整数 `k`。

如果一个 **二进制字符串** 满足以下任一条件，则认为该字符串满足 **k 约束**：

- 字符串中 `0` 的数量最多为 `k`。
- 字符串中 `1` 的数量最多为 `k`。

返回一个整数，表示 `s` 的所有满足 **k 约束** 的子字符串的数量。

```
输入：s = "10101", k = 1
输出：12

输入：s = "1010101", k = 2
输出：25

输入：s = "11111", k = 1
输出：15
```

**提示：**

- `1 <= s.length <= 50`
- `1 <= k <= s.length`
- `s[i]` 是 `'0'` 或 `'1'`。



### 题解

动态规划

`dp[i][j]` 表示字符串下标从 `i` 到 `j` 的子字符串中有多少个 `1`

初始状态：`dp[i][j] = s[j]`

状态转移：`dp[i][j] = dp[i][j - 1] + s[j]`

结果：整个 `dp` 数组中满足 **k 约束** 的元素个数

时间复杂度：$O(n^2)$

空间复杂度：$O(n^2)$

```java
class Solution {
    /**
        dp[i][j] = dp[i][j - 1] + s[j]
     */
    public int countKConstraintSubstrings(String s, int k) {
        int n = s.length();
        int res = 0;
        int[][] dp = new int[n][n];
        char[] chars = s.toCharArray();
        for(int i = 0; i < n; i ++) {
            for(int j = i; j < n; j ++) {
                if(i == 0 && j == 0) {
                    if(chars[0] == '0')
                        dp[0][0] = 0;
                    else
                        dp[0][0] = 1;
                    res ++;
                    continue;
                }
                dp[i][j] = dp[i][j - 1];
                if(chars[j] == '0')
                    dp[i][j] += 0;
                else
                    dp[i][j] += 1;
                if(dp[i][j] <= k || (j - i + 1 - dp[i][j]) <= k)
                    res ++;
            }
        }
        return res;
    }
}
```
