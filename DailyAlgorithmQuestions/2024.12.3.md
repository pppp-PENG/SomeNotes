[3274. 检查棋盘方格颜色是否相同](https://leetcode.cn/problems/check-if-two-chessboard-squares-have-the-same-color/)

给你两个字符串 `coordinate1` 和 `coordinate2`，代表 `8 x 8` 国际象棋棋盘上的两个方格的坐标。

```
输入：coordinate1 = "a1", coordinate2 = "c3"
输出：true

输入：coordinate1 = "a1", coordinate2 = "h3"
输出：false
```

**提示：**

- `coordinate1.length == coordinate2.length == 2`
- `'a' <= coordinate1[0], coordinate2[0] <= 'h'`
- `'1' <= coordinate1[1], coordinate2[1] <= '8'`



### 题解

直接判断

若把字母也当数字（a-h 对应 1-8），那么字母与数字的和的奇偶性与黑白相关

因此判断两个方格颜色是否相同只需要判断两组 `字母 + 数字` 的奇偶性是否相等

时间复杂度：$O(1)$

空间复杂度：$O(1)$

```java
class Solution {
    public boolean checkTwoChessboards(String coordinate1, String coordinate2) {
        return ((int)coordinate1.charAt(0) + (int)coordinate1.charAt(1) + (int)coordinate2.charAt(0) + (int)coordinate2.charAt(1)) % 2 == 0 ? true : false;
    }
}
```
