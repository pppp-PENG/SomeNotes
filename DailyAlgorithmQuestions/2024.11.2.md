[3226. 使两个整数相等的位更改次数](https://leetcode.cn/problems/number-of-bit-changes-to-make-two-integers-equal/)

给你两个正整数 `n` 和 `k`。

你可以选择 `n` 的 **二进制表示** 中任意一个值为 1 的位，并将其改为 0。

返回使得 `n` 等于 `k` 所需要的更改次数。如果无法实现，返回 -1。

```
输入：n = 13, k = 4
输出：2

输入：n = 21, k = 21
输出：0

输入：n = 14, k = 13
输出：-1
```

**提示：**

- $1 <= n, k <= 10^6$



### 题解

比较两数最低位

​	1. 若相同则不用更改

​	2. 若不同

​		2.1 若 ``n`` 最低位为 0 则返回 -1

​		2.2 若 ``n`` 最低位为 1 则更改次数 +1

时间复杂度：$O(\log_2 n + \log_2 k)$

空间复杂度：$O(1)$

```java
class Solution {
    public int minChanges(int n, int k) {
        int tmp1, tmp2, res = 0;
        for(; n > 0 || k > 0; n = n >> 1, k = k >> 1) {
            if(((n ^ k) & 1) == 0) continue;
            if((n & 1) == 0) return -1;
            res += n & 1;
        }
        return res;
    }
}
```
