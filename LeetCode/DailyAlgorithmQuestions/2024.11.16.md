[3240. 最少翻转次数使二进制矩阵回文 II](https://leetcode.cn/problems/minimum-number-of-flips-to-make-binary-grid-palindromic-ii/)

给你一个 `m x n` 的二进制矩阵 `grid` 。

如果矩阵中一行或者一列从前往后与从后往前读是一样的，那么我们称这一行或者这一列是 **回文** 的。

你可以将 `grid` 中任意格子的值 **翻转** ，也就是将格子里的值从 `0` 变成 `1` ，或者从 `1` 变成 `0` 。

请你返回 **最少** 翻转次数，使得矩阵中 **所有** 行和列都是 **回文的** ，且矩阵中 `1` 的数目可以被 `4` **整除** 。

```
输入：grid = [[1,0,0],[0,1,0],[0,0,1]]
输出：3

输入：grid = [[0,1],[0,1],[0,0]]
输出：2

输入：grid = [[1],[1]]
输出：2
```

**提示：**

- `m == grid.length`
- `n == grid[i].length`
- $1 <= m * n <= 2 * 10^5$
- `0 <= grid[i][j] <= 1`



### 题解

枚举法

因为题目要求行回文且列回文，故矩阵既关于中间列对称，又关于中间行对称。可知在非对称轴上的元素都有对应的 `3` 个对称点，即保证这 `4` 个元素相同（这也同时满足了题意要求的”矩阵中 `1` 的数目可以被 `4` **整除**“）

接着讨论对称轴的情况，若 `m`  为奇数，则行对称轴上有一行元素，若 `n` 为奇数，则列对称轴上有一列元素

当 `m`  为奇数且 `n` 为奇数时，中心元素必须是 `0`（以满足题意要求的”矩阵中 `1` 的数目可以被 `4` **整除**“）

分别计算除中心元素外的行对称轴（如有）与列对称轴（如有）上的元素回文情况，并计数矩阵原有回文 `1` 的个数。若矩阵原有回文 `1` 的个数大于 `0` 且无法被 `4` 整除，同时对称轴上无需要翻转的元素，那么只能依靠翻转其中两个 `1` 元素来维持题目要求，即翻转次数 `+2`

时间复杂度：$O(mn)$

空间复杂度：$O(mn)$

```java
class Solution {
    public int minFlips(int[][] grid) {
        int m = grid.length, n = grid[0].length;
        int res = 0;
        for(int i = 0; i < m / 2; i ++) {
            for(int j = 0; j < n / 2; j ++) {
                res += count(grid, i, j);
            }
        }
        int dif = 0, cnt1 = 0;
        if(m % 2 == 1) {
            int i = m / 2;
            for(int j = 0; j < n / 2; j ++) {
                if(grid[i][j] != grid[i][n - 1 - j]) dif ++;
                else if(grid[i][j] == 1) cnt1 ++;
            }
        }
        if(n % 2 == 1) {
            int j = n / 2;
            for(int i = 0; i < m / 2; i ++) {
                if(grid[i][j] != grid[m - 1 - i][j]) dif ++;
                else if(grid[i][j] == 1) cnt1 ++;
            }
        }
        res += dif;
        if(cnt1 != 0 && cnt1 % 2 != 0 && dif == 0) res += 2;
        if(m % 2 == 1 && n % 2 == 1) res += grid[m / 2][n / 2];
        return res;
    }

    public int count(int[][] grid, int i, int j) {
        int m = grid.length, n = grid[0].length;
        int a = grid[i][j], b = grid[m - 1 - i][j], c = grid[i][n - 1 - j], d = grid[m - 1 - i][n - 1 - j];
        int[] cnt = new int[2];
        cnt[a] ++;
        cnt[b] ++;
        cnt[c] ++;
        cnt[d] ++;
        return Math.min(cnt[0], cnt[1]);
    }
}
```
