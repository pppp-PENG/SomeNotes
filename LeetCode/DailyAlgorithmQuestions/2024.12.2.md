[52. N 皇后 II](https://leetcode.cn/problems/n-queens-ii/)

**n 皇后问题** 研究的是如何将 `n` 个皇后放置在 `n × n` 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 `n` ，返回 **n 皇后问题** 不同的解决方案的数量。

```
输入：n = 4
输出：2

输入：n = 1
输出：1
```

**提示：**

- `1 <= n <= 9`



### 题解

N 皇后问题，递归暴力解决

因为 `n` 最高可取到 `9`，考虑到时间要求，需要在 $O(1)$ 的时间内判断某一个位置是否能放皇后

故而可以在每次放完一次皇后时，记录当前的皇后的攻击范围（使用三个集合分别记录列的攻击范围、左上到右下的斜线攻击范围、左下到右上的斜线攻击范围）

考虑到棋盘上的每个格子只有 `处于攻击范围内` 和 `未处于攻击范围内` 两种状态，因此可以用位图存储其状态

时间复杂度：$O(n^2)$

空间复杂度：$O(n)$ 【来源于递归调用使用的栈空间】

```java
class Solution {
    int res;

    public int totalNQueens(int n) {
        solve(0, n, 0, 0, 0);
        return res;
    }

    public void solve(int row, int n, int col, int dia1, int dia2) {
        if(row == n) {
            res ++;
            return;
        }
        for(int idx = 0; idx < n; idx ++) {
            int i = 1 << idx;
            if((col & i) == 0 && (dia1 & i) == 0 && (dia2 & i) == 0) {
                solve(row + 1, n, col + i, (dia1 + i) >> 1, ((dia2 + i) << 1) % (int)1e9);
            }
        }
    }
}
```
