[3233. 统计不是特殊数字的数字数量](https://leetcode.cn/problems/find-the-count-of-numbers-which-are-not-special/)

给你两个 **正整数** `l` 和 `r`。对于任何数字 `x`，`x` 的所有正因数（除了 `x` 本身）被称为 `x` 的 **真因数**。

如果一个数字恰好仅有两个 **真因数**，则称该数字为 **特殊数字**。例如：

- 数字 4 是 **特殊数字**，因为它的真因数为 1 和 2。
- 数字 6 不是 **特殊数字**，因为它的真因数为 1、2 和 3。

返回区间 `[l, r]` 内 **不是 特殊数字** 的数字数量。

```
输入：l = 5, r = 7
输出：3

输入：l = 4, r = 16
输出：11
```

**提示：**

- $1 <= l <= r <= 10^9$



### 题解

枚举

由题可知，特殊数字即为 **质数的平方数**，因此只需要先列出质数表，判断区间内含有多少质数的平方数即可知道有多少特殊数字

时间复杂度：$O(\sqrt{r})$

空间复杂度：$O(\sqrt{r})$

```java
class Solution {
    public int nonSpecialCount(int l, int r) {
        int n = (int)Math.sqrt(r);
        boolean[] notPrime = new boolean[n + 1];
        for(int i = 2; i < Math.sqrt(n) + 1; i ++) {
            if(notPrime[i]) continue;
            for(int j = 2; j <= n / i; j ++) {
                notPrime[i * j] = true;
            }
        }
        int cnt = 0;
        for(int i = 2; i <= n; i ++) {
            if(!notPrime[i] && i * i <= r && i * i >= l) cnt ++;
        }
        return r - l + 1 - cnt;
    }
}
```
