[999. 可以被一步捕获的棋子数](https://leetcode.cn/problems/available-captures-for-rook/)

给定一个 `8 x 8` 的棋盘，**只有一个** 白色的车，用字符 `'R'` 表示。棋盘上还可能存在白色的象 `'B'` 以及黑色的卒 `'p'`。空方块用字符 `'.'` 表示。

车可以按水平或竖直方向（上，下，左，右）移动任意个方格直到它遇到另一个棋子或棋盘的边界。如果它能够在一次移动中移动到棋子的方格，则能够 **吃掉** 棋子。

注意：车不能穿过其它棋子，比如象和卒。这意味着如果有其它棋子挡住了路径，车就不能够吃掉棋子。

返回白车 **攻击** 范围内 **兵的数量**。

```
输入：[[".",".",".",".",".",".",".","."],[".",".",".","p",".",".",".","."],[".",".",".","R",".",".",".","p"],[".",".",".",".",".",".",".","."],[".",".",".",".",".",".",".","."],[".",".",".","p",".",".",".","."],[".",".",".",".",".",".",".","."],[".",".",".",".",".",".",".","."]]
输出：3

输入：[[".",".",".",".",".",".",".","."],[".","p","p","p","p","p",".","."],[".","p","p","B","p","p",".","."],[".","p","B","R","B","p",".","."],[".","p","p","B","p","p",".","."],[".","p","p","p","p","p",".","."],[".",".",".",".",".",".",".","."],[".",".",".",".",".",".",".","."]]
输出：0

输入：[[".",".",".",".",".",".",".","."],[".",".",".","p",".",".",".","."],[".",".",".","p",".",".",".","."],["p","p",".","R",".","p","B","."],[".",".",".",".",".",".",".","."],[".",".",".","B",".",".",".","."],[".",".",".","p",".",".",".","."],[".",".",".",".",".",".",".","."]]
输出：3
```

**提示：**

1. `board.length == 8`
2. `board[i].length == 8`
3. `board[i][j]` 可以是 `'R'`，`'.'`，`'B'` 或 `'p'`
4. 只有一个格子上存在 `board[i][j] == 'R'`



### 题解

直接模拟

先找出车所在的位置，再对上下左右四个方向的可移动区域判断，统计能吃掉的兵的数量

设棋盘边长为 `n`

时间复杂度：$O(n^2)$

空间复杂度：$O(1)$

```java
class Solution {
    public int numRookCaptures(char[][] board) {
        int x = 0, y = 0;
        int cnt = 0;
        for(int i = 0; i < 8; i ++) {
            for(int j = 0; j < 8; j ++) {
                if(board[i][j] == 'R') {
                    x = i;
                    y = j;
                }
            }
        }
        for(int i = 1; i < 8 - x; i ++) {
            if(board[i + x][y] == 'B') break;
            if(board[i + x][y] == 'p') {
                cnt ++;
                break;
            }
        }
        for(int i = 1; i <= x; i ++) {
            if(board[x - i][y] == 'B') break;
            if(board[x - i][y] == 'p') {
                cnt ++;
                break;
            }
        }
        for(int i = 1; i < 8 - y; i ++) {
            if(board[x][i + y] == 'B') break;
            if(board[x][i + y] == 'p') {
                cnt ++;
                break;
            }
        }
        for(int i = 1; i <= y; i ++) {
            if(board[x][y - i] == 'B') break;
            if(board[x][y - i] == 'p') {
                cnt ++;
                break;
            }
        }
        return cnt;
    }
}
```
